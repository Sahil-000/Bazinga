***********************************************************************************************************************************    Welcome to Bazinga - the merchandise store    ***********************


The E-commerce website is all about selling merchandise clothing and accessories of various different themes. It includes clothing for both men and women including summer and winter season clothing. Also we provide accessories such as bags, watches, perfumes and video games collectables for our gaming community.


************************************************************************************************************************************************************************************************************


Technologies used are:-

1) HTML
2) CSS
3) Bootstrap
4) Python
5) Flask (Python web-framework)
6) SQLAlchemy (Python database)
7) Sqlite (database file)

We used HTML, CSS and Bootstrap for the frontend design of the web application and used Python, flask, sqlalchemy and sqlite for the backend purpose of the website.


************************************************************************************************************************************************************************************************************

Pre-requisites Installation:-

1) Python 3.8 (or latest)
2) Visual Studio Code
3) Sqlite DB Browser
4) sqlite
5) Flask
6) SQLAlchemy


************************************************************************************************************************************************************************************************************

Launching the application:-

1) After installing the above requirements, you need to start Visual Studio Code and go to 'File/OpenFolder..'.
2) Browse to '10668497_557_Ass2' Folder and click on select folder.
3) After selecting the folder, you will see the explorer on the left section of the screen which consists of files inside the folder and the code on the remaining section.
4) Click on the 'Website' folder and then you can see all the files under the folder.
5) You will get the hierarchy just like this-

	> Website
		> static
		> templates
		> __init__.py
		forms.py
		models.py
		views.py
		License.txt
		README.txt
		run.py

6) Click on 'run.py' and press Ctrl + F5, Alternatively you can go to 'Run' on the top menu bar and click on 'run without debugging'. The terminal will pop-up and debug mode will be on, you will get a code like this -
